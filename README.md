# MenoBot – Women's Health Companion

MenoBot is a full-stack web application designed to help users track menstrual health, pain symptoms, and receive personalized insights and meal suggestions.

## 🚀 Features
- Cycle tracking dashboard
- Pain logging system
- Smart insights
- Meal recommendations based on cycle phase
- Chatbot support

## 🛠 Tech Stack
Frontend:
- React
- Tailwind CSS

Backend:
- Node.js
- Express

## 📁 Project Structure
- frontend/ → React app
- backend/ → API server

## ⚙️ Setup

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npm run dev
